# Changelog

Todos los cambios notables de este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2025-11-06

### Añadido
- Interfaz gráfica con Electron
- Escaneo del sistema (temporales, inicio, servicios, memoria)
- Limpieza de archivos temporales
- Optimización de servicios no críticos
- Gestión de aplicaciones de inicio
- Liberación de memoria
- Creación de Restore Points
- Modo Dry Run (simulación)
- Sistema de logs detallado
- Documentación completa (README, SECURITY, INSTALL)
- Scripts PowerShell seguros
- Blacklist de servicios críticos
- Interfaz moderna y responsive

### Seguridad
- Validación de permisos de administrador
- Protección de servicios críticos del sistema
- Creación automática de Restore Points
- Logs de auditoría
- Confirmación antes de cambios reales

## [Unreleased]

### Planeado
- Programación automática de mantenimiento
- Perfiles de optimización (Gaming, Trabajo, Ahorro)
- Soporte para Linux y macOS
- Whitelist/Blacklist personalizable desde UI
- Análisis de rendimiento antes/después
- Reversión automática de cambios
- Notificaciones de mantenimiento

## Tipos de Cambios

- `Añadido` para nuevas características
- `Cambiado` para cambios en funcionalidad existente
- `Obsoleto` para características que serán removidas
- `Eliminado` para características removidas
- `Corregido` para corrección de bugs
- `Seguridad` para vulnerabilidades
